# React App Template

Basic React application template, including:
- Babel
- ES6
- Webpack
- Hot Reloader

## Build Your Own Project

```bash
% git clone git@github.com:thehackerati/react-app-template.git my_app
% cd my_app
% npm install
% npm start
% open http://localhost:3000
```
